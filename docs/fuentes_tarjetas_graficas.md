# Fuentes de datos - Tarjetas graficas PcComponentes

## Objetivo

Analizar de donde vamos a extraer los datos de tarjetas graficas antes de programar el scraper.

## Campos a extraer

| Campo | Ejemplo | Fuente en la pagina | Dificultad | Notas |
|---|---|---|---|---|
| modelo | MSI GeForce RTX 4060 Ventus 2X OC | titulo del producto | baja | suele estar en el nombre |
| precio | 299.99 | ficha/listado | baja/media | puede cambiar o aparecer con descuento |
| memoria VRAM | 8GB GDDR6 | especificaciones | media | puede venir dentro del titulo o ficha tecnica |
| GPU | NVIDIA GeForce RTX 4060 | titulo/especificaciones | media | hay que normalizar nombres |
| bus de memoria | 128 bits | especificaciones | media/alta | no siempre aparece |
| ancho de banda de memoria | 448 GB/s | especificaciones | media/alta | no siempre aparece |
| velocidad de transferencia de memoria | 28 Gbit/s | especificaciones | media/alta | no siempre aparece |
| reloj base | 2280 MHz | especificaciones | media/alta | puede aparecer junto al boost |
| reloj boost | 2512 MHz | especificaciones | media/alta | puede aparecer junto al reloj base |
| salidas de video | HDMI, DisplayPort | especificaciones | media | puede estar como lista |
| resolucion maxima | 7680 x 4320 | especificaciones | media/alta | no siempre aparece |
| valoracion | 4.7/5 | ficha producto | baja/media | comprobar formato |
| resenas | texto de usuarios | seccion opiniones | alta | puede tener paginacion |
